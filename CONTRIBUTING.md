# Contributing

Thanks for cloning. Here's how to engage:

## Star first

If this is useful to you, a star helps others find it — and helps me know what's worth maintaining.

## Found a bug?

Open an issue with:
- What you ran
- What you expected
- What actually happened (paste the error)

## Want to extend it?

Open an issue describing what you'd add before writing code. This is a single-maintainer project — PRs land faster when we've aligned on scope first.

## Questions

Use GitHub Discussions. I check in periodically and will respond.
